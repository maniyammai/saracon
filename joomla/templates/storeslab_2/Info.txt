Template settings:

Main Menu module (position-1, position-9, position-10, position-11)
In the administrative back end, navigate to Extensions → Module Manager and open the Main Menu for editing.
Click on the Advanced Parameters. Please set Menu Tag ID to "nav" without a leading space.
Show Sub-menu Items set to yes.
Save!

Bottom Menu modules (position-9 or position-10 or position-11)
In the administrative back end, navigate to Extensions → Module Manager and open the Menu module for editing.
Click on the Advanced Parameters > Set module parameters:
Show Sub-menu Items set to NO.
Save!

Top Menu module (topmenuleft)
In the administrative back end, navigate to Extensions → Module Manager and open the Main Menu for editing.
Click on the Advanced Parameters. Please set Menu Class Suffix to "list" without a leading space.
Show Sub-menu Items set to No.
Save!

Main Menu module or VirtueMart menu module (position-6 / position-7)
In the administrative back end, navigate to Extensions → Module Manager and open the Main Menu / VirtueMart category for editing.
Click on the Advanced Parameters → set Module class suffix to " line1 line1br line1t" with a leading space.

How to activate Isotope Virtuemart product module?
Go to Admin Joomla > Extensions > Module Manager > create new VirtueMart Products module > set Module position to [b]vmproducts[/b] > set Display type products ( featured, random etc... ) > Display Style > list based ul-li
Add Module Class Suffix.

Suffix example:
Random Product module:  Add module Class Suffix [b] random[/b] with a leading space.
Featured Products module:  Add module Class Suffix  [b]featured[/b] with a leading space.
Latest Products module:  Add module Class Suffix [b]latest[/b] with a leading space.
Best Sales module: Add module Class Suffix [b]best_sales[/b] with a leading space.
Recently Viewed Products module: Add module Class Suffix [b]recently_viewed[/b] with a leading space.
Save!


How to activate Font Awesome icons?

You can place Font Awesome icons just about anywhere with the <i> tag.
To increase the size of icons relative to its container, use icon-large, icon-2x, icon-3x, or icon-4x.

Exemple: <i class="icon-camera-retro"></i> for icon-camera-retro
